### Background

### Relationships to Other IGs

This implementation guide (IG) contains CDA templates for the Personal Advance Care Plan and Portable Medical Order documents defined in the FHIR Advance Healthcare Directives Interoperability IG. For detailed information regarding the use cases supported for these document types and additional background on the content contained in each go to https://build.fhir.org/ig/HL7/pacio-adi/index.html
